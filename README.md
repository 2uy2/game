# HAIBAZO React Entrance Test

React implementation of the point-clearing gameplay from `HAIBAZO_React_EntranceTest.mp4`.

## Features

- Enter a point count and restart the board.
- Click numbered points in ascending order.
- Wrong point click shows `GAME OVER`.
- Correct point click fades the point out before removing it.
- `ALL CLEARED` appears only after every point has disappeared completely.
- Timer starts on restart and stops when the game is cleared or failed.

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm test
npm run build
```
